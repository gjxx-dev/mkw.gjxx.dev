## 💡 常见问题

### Q: 真的完全免费吗？
**A:** 是的！我们不收取任何费用，这是一个练手项目，帮助优质的非盈利项目搭建官网。

### Q: 需要多长时间？
**A:** 通常 1-2 周内完成，具体时间取决于项目复杂度和当前排期。

### Q: 可以使用自己的域名吗？
**A:** 可以！我们支持绑定自定义域名，提供相关配置指导。

### Q: 网站所有权归谁？
**A:** 网站所有权归项目方，我们会提供 GitHub 仓库权限，你可以随时修改和管理。

### Q: 支持哪些功能？
**A:** 支持静态网站的所有功能，包括展示、介绍、下载链接、联系方式等。复杂的后端功能需要具体讨论。

### Q: 不满意可以修改吗？
**A:** 可以！我们会根据你的反馈进行调整，同时你也可以通过 GitHub 仓库自行修改。

### Q: 项目审核标准是什么？
**A:** 主要看项目是否合法合规、是否为非盈利性质、内容是否健康向上。
